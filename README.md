# dwmstatus
how to run
```
git clone https://github.com/haydfree/dwmstatus.git
cd dwmstatus 
make 
./dwmstatus <network interface>
```

### dependencies
- dwm
- on linux: iw, pulseaudio
- on openbsd: none
